## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a
security vulnerability in our project, please report it to us as soon as possible.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please email: [your-security-email@example.com]

We will acknowledge receipt of your vulnerability report within 48 hours and will
send a more detailed response within 7 days indicating the next steps in handling
your report.

We will try to fix the vulnerability as soon as possible. Once we've addressed
the issue, we'll publish a security advisory and credit you (if desired).
